¿Cómo usar Git Ignore?

Paso 1: Crea un archivo llamado .gitignore en la raíz de tu repositorio.
Paso 2: Abre el archivo y agrega los archivos o directorios que deseas ignorar, usando patrones.
Paso 3: Guarda los cambios y realiza un commit del archivo .gitignore.
Paso 4: Si ya hay archivos en el repositorio que deberían estar ignorados, elimínalos del índice con git rm -r --cached .
